# Lime Wallet — Mobile Client (Expo React Native)

This directory contains the mobile application codebase for **Lime Wallet** supporting **iOS and Android**, built using **Expo and React Native**.

---

##  Getting Started

Ensure you have Node.js installed, then execute the following commands in your terminal:

1. **Navigate to the mobile directory**:
   ```bash
   cd mobile
   ```

2. **Install project dependencies**:
   ```bash
   npm install
   ```

3. **Start the Expo server**:
   ```bash
   npx expo start
   ```

---

##  How to Run on Devices

After launching `npx expo start`, a QR code will display in your terminal:

*   **Expo Go App (Physical Phone)**:
    *   Download the **Expo Go** app from the iOS App Store or Google Play Store.
    *   Scan the QR code in the terminal using your phone camera (iOS) or the Expo Go scan tool (Android) to load Lime Wallet instantly!
*   **iOS Simulator**: Press `i` in the terminal logs.
*   **Android Emulator**: Press `a` in the terminal logs.
*   **Mobile Web Preview**: Press `w` in the terminal logs.
